## 六、v3.0核心架构原则

| 原则 | 说明 |
|------|------|
| 脱敏标记隔离 | 案情数据（包含真实信息）→ 证据列表（真实信息）→ LLM Prompt（真实信息） |

| 数据契约 | 证据列表结构化输出，每份证据有关键信息字段 |
